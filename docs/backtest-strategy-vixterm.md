# 回测策略:QQQ 核心 + VIX 期限结构择时(VX1−V3 替代 F&G)

日期:2026-07-05  ·  状态:待实现(策略已过一轮 Codex 审 + 本地数据 sanity check)

## 0. 目标与基准 · 定位

验证一个逆向择时叠加层能否跑赢**纯持有 QQQ**:
- 平时持 QQQ;
- **恐慌**时把一部分仓位换成 TQQQ 抄底,情绪**恢复**时换回 QQQ;
- **贪婪/自满**时上 **VIX 期权**保险。

**基准 = 买入持有 QQQ(全期)。** **定位 = 策略体检 / 假设验证,不是可直接下实盘的结论。**
最关键的不是收益数字,而是锁死三件事:**复权价、无未来函数、次日执行**。

信号从 Fear&Greed 改为 **VIX 期限结构 VX1−V3**:F&G 只到 2021,VX1/VX3 有 2018 起,能覆盖 2020 COVID 完整的恐慌→恢复;且一条信号天然同时给恐慌(backwardation)和贪婪(steep contango)两端。

## 1. 信号:VX1−V3(唯一择时信号,v1)

`spread_t = VX1_t − VX3_t`(近月 − 第三近月 VIX 期货结算价,点)。
- `spread > 0`(**backwardation**)= 近期恐慌结构化,常在**底部**附近。
- `spread < 0`(**contango**,正常态);**深度 contango** = 极端自满,常在**顶部**附近。

本地实测(2138 个对齐交易日,2018-01-02 → 2026-07-02):spread 区间约 **−7.35 ~ 27.75**,`spread > 0` 共 **380 天**——信号样本充足。

### 1.1 无未来函数(最关键 · 精确定义)
- 前 `warmup = 252` 交易日**不产生信号**(预热门槛)。
- 从第 253 天起,每个交易日 `t` 的 `rank_t = percentileRank(spread[数据起点..t], spread_t)`,
  即用**扩张窗口**、只含截至 `t` 的数据、**包含当天收盘 `spread_t` 本身**。
- warmup 只是"信号不激活"的门槛,**不丢弃 warmup 之前的数据**;扩张窗口左端始终是数据起点。
- 绝不能用全历史(未来)分位。

### 1.2 触发规则(带滞后 hysteresis)
| 状态 | 进入 | 退出 |
|------|------|------|
| **恐慌**(→ 加 TQQQ) | `spread_t > 0` **或** `rank_t ≥ 85` | `rank_t ≤ 50` |
| **贪婪**(→ 上保险) | `rank_t ≤ 10` | `rank_t ≥ 30` |

恐慌与贪婪互斥。**必做消融**(否则不知收益来自 backwardation 还是滚动高分位):
- **A**:恐慌进入仅 `spread_t > 0`
- **B**:恐慌进入仅 `rank_t ≥ 85`
- **C**:A OR B(默认)

## 2. 仓位映射
- 无信号:**100% QQQ**。
- 恐慌:sleeve `S`(默认 **30%**)QQQ → **TQQQ**;退出换回。
- 贪婪:保险腿(见 §5)—— v1 用**现金减仓**(baseline),v2 用**合成 QQQ 看跌期权**。
- 两端互斥,不叠加。

## 3. 执行细节
- 信号用当日**收盘**(VIX 期货 EOD 结算)算,**次日执行**。
- v1 采用**信号日收盘生成 → 次日 close-to-close 持有收益**(只需复权收盘价,干净可复现);
  **文档不声称"次日开盘成交"**。若要开盘成交,需复权**开盘价**(Yahoo 的 adjusted open 需另取),届时再加并在报告注明。
- 成本:每次换手单边 **5 bps**(可调)。
- 一律用**复权价**(见 §6 的 TQQQ 复权硬要求)。日频。

## 4. 分腿 + 消融
1. 纯持 QQQ(基准)
2. QQQ + **仅轮动腿**(TQQQ)
3. QQQ + **仅保险腿**
4. 两腿全开
外加 §1.2 的恐慌进入 A/B/C 三种,看清每条腿与每种触发条件各自贡献。

## 5. 保险腿 = 合成 QQQ 看跌期权(σ=VXN),不是 VIX 期权/VXX

**为什么不直接回测 VIX 期权**:难点不在缺价格,而在期权**定价的 IV 输入**——VIX 期权的 IV 是"波动率的波动率",只能用 VVIX 近似还得忽略很陡的 smile,误差大。VXX 更糟(短期 VIX 期货 ETN,自带 roll/contango 损耗,叠进另一个策略)——**明确不用**。

**替代:合成 QQQ 看跌期权。** 保险的目的是给 QQQ 底仓上崩盘凸性保护——直接用 **QQQ put** 反而最对口、也最可回测:
- **IV 输入直接观测**:指数看跌期权的隐含波动率**就是 VXN**(纳指 30 天隐含波动率),不用猜。这是相对 VIX 期权的本质优势。
- **凸性保留**:put 崩盘时非线性赔付,正是"保险"要的(现金减仓给不了)。
- **数据全有**:QQQ(`price_eod`)+ VXN(`market_series`)+ 一个 Black-Scholes 纯函数。

**机制**:贪婪激活时,买 ~30 天 QQQ put,strike = ATM 或 ~5% OTM(参数);Black-Scholes 定价,`σ = VXN/100`、`r ≈ 0`。

**头寸规模用两个参数**(避免"拿 20% NAV 去买权利金"这种烧穿式误读):
- **保护名义比例**(默认 **20%**):put 覆盖组合价值的 20%(决定合约数)。
- **权利金预算**(默认 **2% NAV/年**):权利金由 BS 算出、**从现金扣**;若某次 roll 的权利金超过预算按月摊分,则按比例缩减保护名义。

**估值(mark-to-market)**:持有期间**每日按 BS fair value 计入组合净值**;贪婪退出或月度 roll 时按当日 fair value 平仓/换仓;若持到到期,按 intrinsic `max(K − QQQ, 0)` 结算。**不是只在到期看 payoff。**

**Caveats**:
1. **skew**:指数 put 有 skew(虚值 put 的 IV 高于 ATM 的 VXN)→ 用 VXN 平价会**略低估权利金、结果偏乐观**;加 skew 加价(`σ×1.1` 或 +N vol pts)近似修正,并注明是近似。
2. **标的不同**:实盘你可用 VIX 期权;但回测用 QQQ put 作**可处理的凸性代理**——两者标的不同(QQQ put 对冲 QQQ 下跌;VIX 期权押波动率尖峰),结论只对"崩盘时给 QQQ 上凸性保护"这个**概念**有效,不等于 VIX 期权 P&L。

**对照基准 = 现金减仓**(v1 就有):贪婪时把 sleeve `H` 转现金——看 QQQ put 的凸性到底比单纯减仓多换回多少回撤保护、值不值权利金。

**可选零建模代理**:Cambria **TAIL** ETF(2017+,Yahoo 有价,内部就是"国债+滚动 SPX put"),贪婪时买它——零期权建模,但标的是 SPX、有管理费,只算粗代理。

## 7. 关键陷阱与诚实声明
1. **样本:重大恐慌事件 3–4 轮,但状态机会切出很多小 episode。** 本地按默认阈值实测:
   panic **474 天 / 34 个 episode**、greed **446 天 / 35 个 episode**。报告须同时给"重大事件轮数"和"episode 数",别混淆。
2. **无未来函数**:§1.1 的扩张窗口 + 预热 + §3 次日执行。**要有单测锁死这条**(喂一段数据,断言 `rank_t` 不含未来点)。
3. **点位时**:VIX 期货/VVIX 结算不修订,干净;QQQ/TQQQ 用复权价。
4. **TQQQ 用真实复权价**,不要 3×QQQ 模拟(真实 TQQQ 自带每日再平衡衰减 + 费用)。
5. **保险腿用合成 QQQ put(σ=VXN)**,不是 VIX 期权也不是 VXX(见 §5)。两个 caveat:put skew 使权利金偏低估→加 skew 加价;QQQ put ≠ VIX 期权,只对"给 QQQ 上崩盘凸性"概念有效。现金版是对照基准。

### 初步 sanity check(本地粗跑,非正式结果)
"不抓 TQQQ、只做 greed 20% 现金":纯 QQQ CAGR ≈ **20.25%** / MDD **−35.12%**;greed-现金版 CAGR ≈ **20.51%** / MDD **−33.19%**,加 5 bps 换手后期末净值仍略高。**不能当结论**,但说明减仓腿不是一眼没戏。

## 9. 分期落地
- **v1(先做)**:QQQ 基准 + **TQQQ 轮动腿**(A/B/C 消融)+ **现金减仓保险基准**。
  先补 TQQQ 复权价、写日频回测器(信号/撮合分离,纯函数为主,无未来函数有单测)。
- **v2**:**合成 QQQ put 保险腿**(Black-Scholes,σ=VXN,月度 roll,skew 加价);对比现金基准看凸性值不值权利金。数据已齐(QQQ+VXN),只需加 Black-Scholes 纯函数。
- **v3(可选)**:若日后拿到真实 VIX 期权历史,再做原策略的 VIX 期权版。

## 10. 实现建议
日频回测器:遍历交易日 → as-of 数据算 `spread`/`rank`/状态 → 定目标权重 → 用次日复权 close-to-close 收益推进净值 → 记录 equity/交易/指标。信号与撮合分离,便于对"无未来函数"和"复权收益"各写一个锁死单测。
