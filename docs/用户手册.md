# My Trading View 用户手册

这是一款**只在你自己电脑上运行**的金融市场看板，帮你每天用一眼时间看清美股（以及利率、日本、加密等）市场处于"恐慌、平静还是过热"。

## 它能为你做什么

- **每天自动给你抓取关键市场数据**：不用打开十几个网页一个一个看，开机就能看到最新的全市场状态
- **一张图看完恐慌指数、利率曲线、信用利差、油市、日本、比特币**：所有你关心的维度分门别类，互相对照
- **完整保留多年历史**：多数指标能拉十几年甚至几十年（VIX 到 1990、美元指数到 1971），随手对比当前和 2008、2020 那种危机时刻
- **零订阅费用、数据不出你电脑**：不用付 Bloomberg、不用 TradingView Pro，所有原始数据存在你自己的硬盘
- **专门给"看趋势"优化**：周末/假期自动隐藏空白，把图表压紧，跟专业终端的视觉一致

---

## 界面怎么逛

看板分**两级导航**：

- **左边一竖排 = 视角（Perspective）**：期权 / 信用 / 流动性 / 情绪 / 宏观 / 能源 / 利率 / 日本 / 信用曲线 / 通胀 / 估值 / 特色指标。点一个切换大主题。
- **上面一横排 = 子 Tab**：有的视角内部还分几个小页（比如"利率"里有收益曲线、期限走势、SOFR OIS、利率波动率）。只有一个页的视角（比如"信用"）不显示横排。

右上角还有 **5 个时间粒度按钮** 和一个 **数据状态灯**（见后文）。

## 各视角详细介绍

### 期权（Options 25Δ）

**是什么**：每天追踪一批标的的 25-delta 看涨/看跌期权隐含波动率，以及它们的差值（偏度 skew）。横排 Tab 覆盖 SPY / QQQ / VIX / TLT / GLD / USO（来自本地 moomoo）和 BTC（来自 Deribit）。有免费波动率指数的标的（SPY/QQQ/GLD/USO/BTC）还多两块 pane：**隐含 vs 已实现波动率** + **VRP（波动率风险溢价）**。

**能帮你做什么**：判断市场对"上涨"和"下跌"的恐惧不对称程度，以及"卖波动率划不划算"。
- SPY 偏度（put IV 减 call IV）大 = 大家在抢着买跌保护
- VRP 长期为正 = 隐含一直比实际贵，卖方（sell vol）有溢价可赚

> ⚠️ 这个视角的 25Δ 序列从你**第一次运行**那天开始积累，没有历史（行权价每天滚动，补不回来）。一个月后才有 20 个点，三个月后才看得出形态。

### 情绪（Sentiment）

**是什么**：两个横排子 Tab。
- **波动率**：把"恐慌指数家族"塞在一起——VIX、VXN、隐含相关性 COR1M、成分股波动率 VIXEQ、VIX 期货期限结构（VX1−VX3）、以及 Cboe 风险逆转指数 RXM/SPX。
- **情绪**：CNN 恐惧贪婪指数（Fear & Greed）。

**能帮你做什么**：判断市场情绪冷热。
- VX1 比 VX3 低（backwardation）= 当下极度恐慌
- 恐惧贪婪指数跌到极端低位往往是阶段性底部信号

### 信用 / 流动性（Credit / Liquidity）

**是什么**：宏观"水位"指标，来自美联储 FRED。
- **信用**：高收益债信用利差（HY OAS）——企业借钱的"风险溢价"。
- **流动性**：净流动性（美联储资产 − 财政部账户 − 逆回购）、逆回购规模、回购用量、回购压力（IORB−SOFR）。

**能帮你做什么**：信用利差飙升 = 市场开始担心违约；净流动性收缩通常压制风险资产估值。

### 宏观（Macro）

**是什么**：增长 / 通胀 / 政策的综合 regime 读数（薪资增速、黏性 CPI、利率等）。

**能帮你做什么**：一眼看清当前处在"过热 / 滞胀 / 衰退 / 复苏"的哪个象限。

### 能源（Energy）

**是什么**：油市结构指标——Brent 减 WTI 价差、柴油裂解价差（diesel crack）。数据走 Yahoo 近月期货。

**能帮你做什么**：Brent−WTI 走阔常反映海运/地缘扰动；柴油裂解走高说明炼油端/物理需求紧张，是实体经济的温度计。

### 利率（Rates）

**是什么**：4 个横排子 Tab。
- **收益曲线**：美债 1M–30Y 全曲线（今天长什么样）。
- **期限走势**：10Y、3M 及其利差（10Y−3M）随时间怎么走。
- **SOFR OIS**：无风险利率互换曲线（市场给美联储路径定的价），来自 Eris。
- **利率波动率**：MOVE 指数（债市版 VIX），来自 Yahoo。

**能帮你做什么**：看美联储紧缩还是放水、看"收益率倒挂"（短端高过长端预示衰退）、看债市有多慌。

### 日本（Japan）

**是什么**：4 个横排子 Tab——USD/JPY 汇率 + CFTC 日元投机净持仓、JGB（日本国债）收益曲线、JGB 期限走势、日债波动率（JGB VIX）。

**能帮你做什么**：跟踪日元套息（carry trade）风险。日元急升 + 持仓反转往往伴随全球去杠杆（2024-08 那种）。

### 信用曲线（Credit curve）

**是什么**：两个子 Tab——按评级的信用利差（AAA→CCC 梯队）、按期限的信用利差结构。

**能帮你做什么**：看风险偏好在评级/期限维度上怎么分布，比单一 HY 利差更细。

### 通胀（Inflation）

**是什么**：3 个子 Tab——通胀预期 BEI（盈亏平衡通胀，名义国债减 TIPS）、通胀走势（10Y、5Y 及利差）、通胀来源（含 RBOB 汽油同比等前瞻项）。

**能帮你做什么**：市场给通胀定的价（BEI）领先于官方 CPI；汽油同比是最快的通胀前瞻信号之一。

### 估值（Valuation）

**是什么**：席勒 CAPE（周期调整市盈率，Robert Shiller 数据集，回溯到 1871）。

**能帮你做什么**：判断整体股市贵不贵。CAPE 处在历史高位说明未来长期回报预期偏低。

### 特色指标 → 攻防（Featured）

**是什么**：NOBL（红利贵族）÷ QQQ（科技）比值，配 ZigZag 算法自动标出"进攻 / 防守"阶段切换。

**能帮你做什么**：一眼看清市场风格——资金在往防御性红利股躲（防守），还是在追高 beta 科技（进攻）。

## 通用操作

### 时间粒度切换

右上角 5 个按钮：**1D / 1W / 1M / 1Q / 1Y**。

这是**显示粒度**（每根 K 线代表多长时间），不是时间范围——图默认就展示全部历史，切粒度只是把多天聚合成一根，让长历史看着不拥挤。

- **1D**：原始日线，每根线一个交易日
- **1W / 1M / 1Q / 1Y**：把多天聚合成一根（取该周期最后一天的收盘价），越大越"缩略"

切换是全局的——切到 `1M`，所有视角的图都会一起变。

### 在指标之间对照

每个 pane 里多条线**默认同图**，颜色不同。鼠标在图上移动会显示十字光标和每条线在当前日期的具体数值。

💡 小提示：同一张图里量纲不同的序列会分到左右两轴（比如百分点的利率和指数点位各走一边）。鼠标停下来读数值才是准的。有分布的序列还会画 P5/P95 分位带 + 当前分位徽章，极端区间用红（风险端）/ 绿（机会端）底色高亮。

### 每个 pane 右上角的 ⓘ

鼠标悬停 ⓘ 图标会弹出这条指标的说明（是什么、怎么读），不用记。

### 数据状态灯

顶栏左上角有个小圆点：

- 🟢 **绿色**：最近一次后台数据拉取一切正常
- 🟡 **黄色**：部分数据源失败（比如有几个 ticker 没拉到）
- 🔴 **红色**：完全失败（可能是网络问题或者 FRED key 失效）
- ⚪ **灰色**：还没运行过

鼠标悬停可以看具体哪个数据源出问题。

## 使用步骤 / 操作指南

### 如何第一次启动看板

1. 打开 terminal，进入项目目录
2. 输入 `bun install` 等它装完依赖
3. 去 [FRED 网站](https://fred.stlouisfed.org/docs/api/api_key.html) 免费申请一个 API key（30 秒）
4. 复制 `.env.example` 为 `.env`，把 key 填进去
5. 输入 `bun run db:migrate` 建数据库
6. 输入 `bun run job:daily` 第一次拉数据（含历史，要等几十秒到一两分钟）
7. 输入 `bun run dev`——一行命令同时跑前端和后端
8. 浏览器打开 `http://localhost:5173`

💡 小提示：第 7 步的 `bun run dev` 会在一个 terminal 里同时启动前端和后端，按 Ctrl-C 一次性关闭两个。

### 如何让看板每天自动更新

自动更新已经配好了（两个后台定时任务，白天每隔几小时各跑一次）。你不用做任何事，当天只要成功拉到一次就自动停，不会重复跑：

- **股票组**（含期权、VIX 期货、Eris SOFR OIS）：需要本机 OpenD 开着。
- **BTC 组**：每天跑（BTC 7×24 小时交易，周末也有行情），不需要 OpenD。

- 想看最近几天有没有正常跑：terminal 里输入 `./scripts/cron.sh history`
- 想立刻手动拉一次：`bun run job:daily`（股票）/ `bun run job:crypto`（BTC）

💡 小提示：多数 regime 视角（信用/流动性/宏观/能源/利率/日本/通胀/估值）是**打开页面时实时拉取**的，不依赖定时任务，也不占存储；只有期权、VIX 期货、Eris、BTC 这几类落库的才靠 job。

### 如何切换看不同的指标

1. 左边竖排点视角（期权 / 信用 / …），上面横排点子 Tab
2. 切换时图表瞬间替换，时间粒度保持不变
3. 想看更粗的长历史就点右上角 `1M` / `1Q`

### 如何手动重新抓一次数据

如果你想立刻拿到最新数据（不等明天），直接：

```
bun run job:daily        # 股票 / 期权 / VIX 期货 / Eris
bun run job:crypto       # BTC
```

跑完会显示完成提示，刷新浏览器就能看到。

## 常见问题 FAQ

**Q：这个工具是免费的吗？**
A：是。所有数据源（moomoo 本地行情、Deribit、CBOE、FRED、Yahoo、Eris、日本 MOF/JPX、CFTC、Shiller、CNN）都免费。你不需要付任何订阅费。

**Q：我的数据存在哪里？会被人看到吗？**
A：落库的数据都存在你电脑里的一个文件 (`data/mtv.db`)。这个工具默认不上传任何东西到外面，也不需要登录账号。

**Q：FRED 那个 API key 安全吗？**
A：放在 `.env` 文件里，这个文件已经在 git 忽略列表中，不会被提交到 GitHub。即使万一泄露，FRED 的 key 也只能用来读公开经济数据，吊销重发也是 30 秒。

**Q：周末/假期看板会显示什么？**
A：周末美股不交易，所以看板显示的是最近一个交易日的数据（通常是周五）。X 轴上不会出现周六周日的空白，这跟专业终端的做法一致。

**Q：我电脑关机/休眠了一晚，第二天数据会不会缺一天？**
A：**大部分数据会自动补齐**，但期权例外。详见下一条。

**Q：连续关机好几天会发生什么？**
A：分两种情况：

- **实时拉取的视角（信用/流动性/宏观/能源/利率/日本/通胀/估值等）**：✅ 不受影响，它们每次打开页面都现拉全历史，跟你关没关机无关。
- **落库的股指 / VIX 期货 / Eris**：✅ 能补。下次开机会拉"上次成功日 ~ 今天"之间缺失的工作日。
- **25Δ 期权**：❌ **不能补**。期权源（moomoo / Deribit）只提供"当前快照"，查不到"3 天前的期权链"。你关机那几天的期权点永久丢失，曲线上显示成断点。

💡 小提示：如果你长期出差关电脑，出差前手动跑一次 `bun run job:daily` 拿到当天数据。但回来后中间那几天的期权数据无论如何补不回。

**Q：可以在手机上看吗？**
A：目前不行——这个工具是给本地电脑（macOS）设计的，而且期权还依赖本机的 moomoo OpenD。

**Q：能加我自己关心的指标吗（比如美元/日元、纳指、某个 ETF）？**
A：能，而且很省事。项目按"配置驱动"设计——加一个期权标的就往 `MARKET_CATALOG` 加一条，加一个 regime 指标就往 `REGIME_DIMS` 加一条，加一个视角/子 Tab 就往 `PERSPECTIVES` 加一条，其它地方不用动。告诉懂代码的人改一处即可。

**Q：为什么期权 Tab 几乎是空的？**
A：期权数据源只给"当前快照"，没有历史。从你第一次启动那天开始累积，每天加一个点。一个月后才会看出短期形态，建议先放着养着。

**Q：图表为什么看起来 Lag/迟钝？**
A：通常不会，但如果 `1D` 视图同时显示几万个数据点 + 很多条线，浏览器有点压力。换到 `1M` 或 `1Q` 就立刻流畅。

**Q：状态灯一直红的，怎么办？**
A：鼠标停上去看具体哪个数据源失败。常见原因：
- `macro: FRED_API_KEY is required` → 你忘了配 `.env` 里的 FRED key
- 某个 Yahoo/Eris 源临时抽风 → 等一天或重跑对应 job；这些源都做了优雅降级，单个挂掉只会让那一块 pane 缺失，不影响其它
- 全部失败 → 你电脑没联网

**Q：可以多久不维护？**
A：理论上可以一直放着自动跑。但 Yahoo、Eris、CNN 这类免费/爬虫源偶尔会变接口，到时候可能需要更新一下对应库版本或 fetcher。

## 应用场景举例

### 场景 1：开盘前 5 分钟的状态扫描

小李是一位**业余美股投资者**，每天早上 8 点（日本时间）起床想看看"昨夜美股发生了什么"，但又懒得打开 5 个网站。

以前他要分别查 VIX、查 10 年期美债、看一眼 BTC、刷一遍信用利差——光是开标签就要 3 分钟。

用了 `My Trading View` 之后，他打开一个本地链接，**竖排视角一个个点过去**，VIX 突破 20、10Y 利率有没有跳、HY 利差有没有走阔，一目了然。**整个流程从 3 分钟压缩到 30 秒**。

### 场景 2：市场大跌时的"历史对比"

老王是一位**有 10 年经验的交易员**，某天美股突然跳水 4%，他想知道"这次跟 2020 年 3 月、2008 年 10 月相比是大事还是小事"。

免费的网页版行情最远常只能看 5 年——比不上 2008。

用 `My Trading View` 把粒度切到 `1M`、拉满历史，**VIX 历史最高的两根尖刺**（2008-11-20 的 80.86 和 2020-03-16 的 82.69）**一目了然**——他今天的 VIX 才到 25，跟这两根比还差得远。**他放心了，没有清仓**。

### 场景 3：跟踪日元套息风险

阿强关注**全球去杠杆的传导**。他每天扫一眼"日本"视角：USD/JPY 有没有急升、CFTC 日元净持仓有没有从大幅净空快速回补、日债波动率（JGB VIX）有没有跳。

2024 年 8 月那波，正是日元急升 + 空头回补 + 日股闪崩连锁引发了全球风险资产回撤。**有了这个视角，他能在同一屏里把"日元—美债—美股"串起来看**，而不是分散在三个网站里各看各的。
